## Installation

1. 安装RabbitMQ延时扩展`rabbitmq_delayed_message_exchange`
[插件地址](https://github.com/rabbitmq/rabbitmq-delayed-message-exchange/releases/tag/v3.8.0)

2. 安装hyperf-rabbitmq延时组件
```bash
composer require zerduo/rabbitmq-delay
```
